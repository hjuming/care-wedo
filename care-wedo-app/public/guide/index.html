Care WEDO Guide

第一次使用 Care WEDO

這頁是不用 JavaScript 也能讀取的使用教學。目標是讓長輩只做主要功能，設定、邀請與資料確認由照護協作者代勞。

首頁 常見問題 收費說明 進入後台

1. 登入 Care WEDO

家人先用 LINE Login 登入 Care WEDO 後台。若之前登入過，除非使用者登出，回到首頁後應直接進入後台首頁，減少長輩重複登入。

2. 加入 Care WEDO LINE 小管家

若要收到上傳摘要、每日提醒與晚間空腹提醒，家人需要加入 Care WEDO LINE 小管家。LINE Login 不等於加入官方帳號。

3. 建立家庭群組與照護對象

每個家庭群組預設由主帳號管理。建議同一群組最多 4 位主要照護對象、5 位共同協作者、1 位主帳號，避免資料混亂與操作負擔。

4. 上傳醫療資料

用手機拍看診單、藥袋、處方箋或預約單，也可以貼上醫療相關文字。系統會先整理成待確認資料，再由家人確認後存入照護紀錄。

5. 長輩只看主要提醒

今日照護只保留拍照新增、查看提醒與完成確認。編輯照護對象、手動新增提醒、邀請協作者、費用預估與通知設定，都集中在協作者管理中心。
